# 🚀 Projeto CRUD: Sistema de Estoque & Vendas Simulado

Este projeto foi desenvolvido como material de estudo para o meu canal no **YouTube**. O objetivo é mostrar como criar uma aplicação funcional do zero, passando por todas as etapas fundamentais que um programador iniciante precisa dominar.

## 📺 O que você vai aprender neste vídeo
Neste tutorial, eu guio você pelo seguinte caminho:
* **Setup do Ambiente:** Instalação e configuração do **VS Code**.
* **Versionamento:** Criação de conta no **GitHub** e uso do **GitHub Desktop** para gerenciar o código.
* **Desenvolvimento:** Criação de um CRUD completo (Cadastro de Produtos) com armazenamento local (**LocalStorage**).
* **Lógica de Negócio:** Simulação de uma aba de vendas com baixa de estoque automática e checkout de cartão.
* **Deploy:** Como colocar o seu site no ar de graça usando o **GitHub Pages**.

---

## 🛠️ Tecnologias Utilizadas
* **HTML5** (Estrutura)
* **Tailwind CSS** (Estilização moderna e rápida)
* **JavaScript ES6+** (Lógica de manipulação de dados e DOM)
* **LocalStorage** (Banco de dados interno do navegador)

---

## 📌 Funcionalidades do Projeto
1.  **Aba de Cadastro:** * Upload de foto do produto.
    * Campos de nome, preço, descrição e quantidade.
    * Listagem e exclusão de itens.
2.  **Aba de Vendas:**
    * Vitrine interativa.
    * Seleção de itens para checkout.
    * Simulador de pagamento com cartão de crédito.
    * Atualização de estoque em tempo real.
